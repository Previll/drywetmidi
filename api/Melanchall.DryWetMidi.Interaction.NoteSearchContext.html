    Fields

    Name Description
    AllEventsCollections A note can be detected within all events collection (EventsCollection or TrackChunk) of the source (MidiFile for example). It means MIDI events that make up a note can be present in different events collections.
    SingleEventsCollection A note should be detected only within single events collection (EventsCollection or TrackChunk). It means MIDI events that make up a note should be present in one events collection.
